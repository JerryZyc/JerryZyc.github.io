# 贡献指南

## 文件命名

- 文章必须放在 `_posts/`，命名为 `YYYY-MM-DD-title.md`。
- 标题使用小写字母与连字符。

## Front Matter（头部）

必填字段：

- `title`
- `date`
- `categories`
- `tags`

论文笔记还需包含：

- `paper.venue`
- `paper.year`
- `paper.authors`
- `paper.pdf`
- `paper.code`
- `status`（to-read | reading | done | reproduced）

## 分类

使用以下之一：

- `learning`
- `paper`
- `skills`
- `work`

## 图片

- 图片放在 `assets/images/`。
- 引用使用相对路径，如 `/assets/images/your-image.png`。

## 参考文献

- 在 `## 参考文献` 小节下用列表。
- 草稿阶段可使用占位内容。
